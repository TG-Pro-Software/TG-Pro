## What is TG Pro?
TG Pro is a macOS utility built for users who want detailed insight into their system’s **[temperature monitoring](#)** and cooling behavior. It exposes internal hardware sensors that are normally hidden, giving a clear view of how heat affects system performance.

By tracking values such as **[CPU temperature](#)**, **[GPU temperature](#)**, battery sensors, and drive readings (where supported), TG Pro helps users understand how their Mac behaves during everyday tasks and under sustained heavy workloads.

TG Pro also provides advanced **[fan management](#)** and **[temperature logging](#)**, making it possible to review historical data, diagnose overheating issues, and identify throttling patterns over time.

With real-time monitoring and a convenient **[menu-bar interface](#)**, TG Pro is a practical tool for users who want greater control and awareness of system temperatures on macOS.
